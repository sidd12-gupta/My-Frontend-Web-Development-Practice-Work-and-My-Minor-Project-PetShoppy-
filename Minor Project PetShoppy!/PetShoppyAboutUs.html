About PetShoppy! - Online Pet Accessories Shop

PetShoppy! is an online pet shop website developed by Siddharth Gupta. Now, it is mainly a frontend focused website and in the later or major stages of development it will use backend technologies to increase its functionality.This is an online pet shop website that helps users who are looking for decent accessories for their pets.

The tools used for making this website are not using any out of the world technology. Microsoft Visual Studio Code is an IDE which is used for implementing our various HTML,CSS and JavaScript codes and the backend part will also be used on this IDE. The second tool which most are familiar with and are viewing our site is the web browser for sure. Any web browser can be used for viewing my website.

The main focus of this site is to provide customers, who are having pets, accessories for their pets at an affordable cost. Another objective can be to make this website easy-to-use(clutter-free) and responsive so that it can resize itself according to different screen resolutions.
